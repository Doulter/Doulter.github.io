嘟特接入第三方SDK明细及权限说明

 

为了适配在不同手机设备上使用，保障嘟特相关功能的实现与安全稳定的运行，嘟特接入了第三方提供的SDK。

嘟特所接入的SDK可能会调用您的设备权限、获取您的相关信息；嘟特会对合作方获取信息的SDK进行严格的安全监测，以保障您的信息安全；同时尽可能及时更新并向您说明嘟特所接入第三方SDK的最新情况。

嘟特接入的相关第三方SDK目录如下：

1、EventBus SDK

功能/服务：用户开发者Android事件发布/订阅框架

信息类型：唯一设备识别码（安卓 ID）

调用权限：无

官网链接：https://github.com/greenrobot/EventBus

 

2、ExoPlayer SDK

功能/服务：应用层媒体播放器开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/google/ExoPlayer

 

3、Glide SDK

功能/服务：一个快速高效的Android图片加载的开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/bumptech/glide

 

4、Okhttp3 SDK

功能/服务：用于Android中请求网络的开源框架

信息类型：无

调用权限：无

官网链接：https://square.github.io/okhttp/

 

5、Okio SDK

功能/服务：用于Android中IO操作的开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/square/okio

 

6、RxJava SDK

功能/服务：用于Android中线程调度的开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/ReactiveX/RxJava

 

7、SimpleCropView SDK

功能/服务：用于Android中圆角控件的开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/igreenwood/SimpleCropView
